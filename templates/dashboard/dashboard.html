Questions

One question per line. Questions are single-line text answer questions by default.
Write options seperated by slashes in brackets after the question (e.g. "Do you like the color red? (yes/no)") to create fixed options.
Prepend the question with an asterix (*) to make it required.
Examples:
* What do you think of this?
Do you like coffee? (yes/no)
* Which of these shapes if your favourite? (square/triangle/circle)
